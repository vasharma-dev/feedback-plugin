How it works

Add it in three ways

The same feedback engine, integrated however your product is built — no backend work required.

1 · Zero-code script

Paste one tag. A themed 💬 button appears, bottom-right.

<script src="https://cdn.jicama.tech/feedback.js"
  data-key="pk_your_key"></script>

2 · JavaScript SDK

Full control + attach the logged-in user to every submission.

import { Feedback } from '@jicama/feedback';
Feedback.init({ key: 'pk_your_key' });
Feedback.open();

3 · REST API

Any platform — mobile, desktop, CLI, backend.

POST /v1/feedback
Authorization: Bearer pk_your_key
{ "type":"bug", "message":"..." }
